# Construction Company Database

Курсова робота з дисципліни «Бази даних».

Проєкт містить SQL-скрипти для створення та реалізації бази даних
«Будівельна компанія» у середовищі Microsoft SQL Server.

## Структура репозиторію
- SQLQuery1_TABLE.sql — створення таблиць і зв’язків
- SQLQuery2_VMIST.sql - заповнення даними
- SQLQuery3_CONSTRAINS.sql — обмеження цілісності
- SQLQuery4_ZAPETI.sql — аналітичні SQL-запити
- SQLQuery5_HISTORYTABLE.sql — історичні (temporal) таблиці
- SQLQuery6_GETSTOREDPROCEDURE.sql — Get stored procedures
- SQLQuery7_SETSTOREDPROCEDURE.sql — Set stored procedures
